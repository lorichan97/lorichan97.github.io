Lists Usage
Unordered List Each item is shown, by default, with a bullet - those lists are generally used to present a list of items that not necessarily need to be shown in any specific order. Start an unordered list with an <ul> tag
Ordered List Each item is shown, by default, with a number - those lists are generally used to present a list of items that need to be shown in a specific order. Start an ordered list with an <ol> tag
Definition List The items come in pairs - the term to be defined and the definition (description). This list is used for glossary, vocabulary, questions & answers, etc. Start a definition list a <dl> tag
